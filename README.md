# DragonIR - Go Experimentation Laboratory

## 🔬 About the Project

DragonIR is a black box project developed in Go, dedicated to experimentation, testing, and studying different libraries, implementations, and features of the Go language. This environment serves as a playground to explore new technologies and development practices.

## 🎯 Objectives

- Test different Go libraries and frameworks
- Experiment with new implementations and code patterns
- Evaluate Go language features and functionalities
- Serve as a learning and experimentation environment
- Document findings and best practices

## 🛠️ Technologies

- Go (Golang)
- Go libraries (continuously evolving)
- Various frameworks and tools

## 📚 Project Structure

```
dragonir/
├── src/
│   ├── handlers/     # Request handlers
│   ├── views/        # View layer
│   ├── templates/    # HTML templates
│   ├── router/       # Route configuration
│   └── main.go       # Application entry point
```

## 🚀 How to Use

1. Clone the repository
2. Install dependencies:
   ```bash
   go mod tidy
   ```
3. Run the project:
   ```bash
   go run src/main.go
   ```

## 📝 Notes

This is a continuously evolving project, used for study and experimentation purposes. Not recommended for production use.
